# Install packages
```
npm i
```

# Start local hardhat node
```
npx hardhat node
```

# Compile, depoly and run
On a different terminal:
```
npx hardhat compile
npx hardhat deploy
npm run start
```
Go to https://localhost:8443
Enjoy!
